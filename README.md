## р╕Бр╕▓р╕гр╕Хр╕┤р╕Фр╕Хр╕▒р╣Йр╕Зр╣Бр╕ер╕░р╣Гр╕Кр╣Йр╕Зр╕▓р╕Щ (Installation & Usage)
1. Clone & Setup Environment:
```bash
git clone https://github.com/wiriyabot/RAG-Admin.git
cd RAG-Admin
python -m venv venv
# Windows
venv\Scripts\activate
# macOS / Linux
source venv/bin/activate

```

---

3. Configuration:

```bash

```

```bash
streamlit run app.py
```
